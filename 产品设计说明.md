# 家庭记账本 - 产品设计说明

## 一、产品概述

家庭记账本是一款基于微信小程序的云端记账工具，支持个人记账和家庭共享记账两种模式。用户可以随时记录收支，查看月度统计和趋势分析，也可以创建家庭组与家人共同管理家庭财务。

### 核心亮点

- **双模式记账**：支持极简模式（仅主分类）和详细模式（主分类 + 子分类）
- **家庭共享**：创建或加入家庭组，家庭成员各自的收支自动汇总到家庭统计
- **云端同步**：基于微信云开发，数据实时同步，换手机不丢失
- **多维度统计**：分类占比饼图、近 6 个月趋势折线图、分类排行
- **数据导出**：支持将账单导出为 Excel 文件，可保存到手机或分享给他人

---

## 二、用户角色

| 角色 | 说明 |
|------|------|
| 个人用户 | 记录和管理自己的收支，查看个人统计 |
| 家庭创建者 | 创建家庭组，可邀请成员、移除成员、解散家庭组 |
| 家庭成员 | 加入已有家庭组，参与家庭统计，可主动退出 |

## 三、功能模块

### 3.1 明细（首页）

**Tab 栏第一个页面，用户打开小程序默认展示。**

- 顶部月份选择器，可切换查看不同月份的账单
- 月度总览卡片：显示当月总收入、总支出、结余
- 账单列表：按日期分组展示，每条记录显示分类、金额、备注
- 悬浮"+"按钮：快速进入记账页面
- 长按或点击可编辑/删除已有记录
- 下拉刷新、上滑加载更多

### 3.2 记账

**从首页点击悬浮按钮或编辑已有记录进入。**

- 收支类型切换：支出 / 收入
- 金额输入：支持小数点后两位
- 分类选择：
  - 极简模式：仅选择主分类（如餐饮、交通）
  - 详细模式：选择主分类后展开子分类（如餐饮 → 零食、聚餐）
- 支持新建自定义主分类和子分类
- 日期选择器
- 备注输入（选填，最多 50 字）
- 保存后自动返回首页并刷新列表

### 3.3 统计

**展示月度收支统计数据和图表。**

- 顶部月份选择器
- 收支类型切换：支出 / 收入
- 总览卡片：总金额、笔数、平均值
- 分类占比饼图（Canvas 2D 绘制）
- 近 6 个月趋势折线图
- 分类排行列表（带进度条）
- 如果用户加入了家庭组，统计页面可选查看家庭维度数据

### 3.4 家庭

**管理家庭组，查看家庭统计。**

- 未加入家庭时：显示"创建家庭组"和"加入家庭组"两个入口
- 创建家庭组：输入家庭名称，自动生成 6 位邀请码
- 加入家庭组：输入邀请码加入
- 已加入家庭时展示：
  - 家庭信息卡片（名称、邀请码，点击可复制）
  - 本月家庭收支统计
  - 分类开销分布
  - 成员贡献排行
  - 成员列表（创建者可移除成员）
- 创建者可解散家庭组，普通成员可退出
- **关键逻辑**：
  - 创建/加入家庭组时，该用户的历史个人记录自动关联到家庭组
  - 退出/被移除/解散时，相关记录的家庭关联自动清除

### 3.5 我的

**个人中心和设置入口。**

- 用户信息展示（昵称、头像）
- 记账模式切换：极简模式 ⇄ 详细模式
- 分类管理入口
- 导出账单入口
- 家庭入口（跳转至家庭 Tab）

### 3.6 分类管理

**管理自定义主分类和子分类。**

- 支出/收入类型切换
- 主分类列表：支持新增、编辑、删除
- 点击主分类进入子分类管理
- 预设分类不可删除，自定义分类可删除
- 极简模式下隐藏子分类管理入口

### 3.7 导出账单

**将账单数据导出为 Excel 文件。**

- 选择导出时间范围（开始月份 - 结束月份）
- 点击"导出为 Excel"生成文件
- 生成后点击"下载账单"打开文件预览
- 预览界面右上角菜单可"保存到手机"或"发送给朋友"
- 导出内容：日期、类型、主分类、子分类、金额、备注

## 六、核心业务逻辑

### 6.1 用户初始化流程

1. 用户首次打开小程序 → `login` 云函数被调用
2. 云函数检查 `users` 集合中是否有该 openid 的记录
3. 如果没有：创建用户文档（`_id` = openid），同时初始化预设分类和子分类
4. 返回用户信息给前端，存入 `app.globalData.userInfo`
5. 所有页面在 `onLoad`/`onShow` 时需要 `await app.loginPromise` 确保初始化完成

### 6.2 记账模式

- **极简模式（simple）**：记账时仅选择主分类，不展示子分类
- **详细模式（detailed）**：选择主分类后展开子分类，可选择子分类
- 在"我的"页面可切换模式，切换后不影响历史记录
- 分类管理页面根据当前模式显示/隐藏子分类管理

### 6.3 家庭组完整生命周期

```
                    ┌──────────────────┐
                    │   未加入家庭组    │
                    └───┬──────────┬───┘
              创建家庭组 │          │ 输入邀请码加入
                    ┌───▼──────────▼───┐
                    │   已加入家庭组    │
                    │  （可查看统计）   │
                    └───┬──────────┬───┘
              退出家庭组 │          │ 创建者解散
                    ┌───▼──────────▼───┐
                    │   未加入家庭组    │
                    └──────────────────┘
```

**关键逻辑：**
- 创建/加入家庭组时，该用户所有历史记录自动关联到家庭组（`familyId` 同步更新）
- 退出/被移除/解散时，相关记录的家庭关联自动清除（`familyId` 置空）
- 家庭统计查询调用 `getFamilyStats` 云函数，按 `familyId` 聚合所有成员记录
- 成员列表查询调用 `getFamilyMembers` 云函数（因为客户端无法查询其他用户的 `_openid`）

### 6.4 数据导出流程

1. 用户在导出页面选择时间范围，点击"导出为 Excel"
2. 调用 `exportExcel` 云函数，将账单数据生成 HTML table 格式的 `.xls` 文件
3. 文件上传到云存储，返回 `fileID`
4. 前端通过 `wx.cloud.downloadFile` 下载文件，保存为 `.xls` 后缀
5. 调用 `wx.openDocument` 打开文件预览（`fileType: 'xls'`, `showMenu: true`）
6. 用户在预览界面右上角菜单选择"保存到手机"或"发送给朋友"
